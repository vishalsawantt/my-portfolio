‘Missing Person’ Android Application

Description: The aim of the Missing Person Android Application is to help missing individuals return home.

Technologies Used:

  • Java
  • Android Studio
  • Firebase Database
  • OpenCV Library
  • Google Maps

Tools Used:

  • Android Studio
  • Firebase Panel
  • Chat-GPT

My Role: This is my solo project aimed at addressing a real-time problem. I developed the Missing Person Android Application to help missing individuals return home. In this app, users can create reports for missing persons. On the other side, users who find unidentified individuals or people who appear lost can easily search the app using face recognition powered by OpenCV, a widely-used computer vision library. They can then inform the people who created the reports, facilitating quicker reunions and improving the chances of finding missing individuals

Development Details: I developed the Android application using Java and implemented the backend with Google Firebase Database.
